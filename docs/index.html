Why BEER?

One tool. Your data stays local.

Protein analysis typically means shuttling between a dozen specialised web servers — each with its own format, latency, and privacy concern. BEER replaces that workflow.

🔒

Privacy-first, offline-capable

All classical analysis and AI predictions run entirely on your machine. No sequence leaves your computer unless you explicitly fetch from UniProt or RCSB.

🧠

AI and classical on the same axes

24 ESM2 BiLSTM heads (disorder, signal peptide, transmembrane, active site, PTMs…) displayed alongside classical profiles — with UniProt curated annotations overlaid for direct visual comparison.

📊

Publication-ready, every time

Every graph exports at 200 dpi in PNG, SVG, or PDF. Underlying data exports as CSV or JSON. A built-in Figure Composer assembles multi-panel publication figures without leaving the app.

Capabilities

Classical analysis meets deep learning

⚗️

Classical Analysis14 sections

  • Physicochemical — MW, pI, GRAVY, extinction coefficient, aliphatic index
  • Charge patterning — FCR, NCPR, κ, Ω (Das-Pappu); sliding-window local charge
  • Hydrophobicity — 9 scales (Kyte-Doolittle, Wimley-White, Hessa, GES…)
  • Phase separation — catGRANULE score, LARKS, sticker-spacer model, Uversky plot
  • β-Aggregation & solubility — ZYGGREGATOR hotspots; CamSol profile
  • Proteolytic map — 9 enzymes with peptide masses
  • Linear motifs — NLS, NES, SUMOylation, 14-3-3, KFERQ, and more
🤖

AI Predictions24 heads

  • Architecture — ESM2-650M → 2-layer BiLSTM; focal loss; MMseqs2-clustered splits
  • Disorder — AUROC 0.991, trained on DisProt curated regions
  • Signal peptide — AUROC 0.9999; includes GPI anchor detection
  • Transmembrane — BiLSTM-CRF with Viterbi topology decoding (outside→TM→inside); AUROC 0.992
  • Functional sites — active site, binding site, DNA/RNA binding (BioLiP) — model weights in training
  • PTMs ×6 — phospho, ubiquitin, acetyl, methyl, glyco, lipid (dbPTM) — model weights in training
  • MC-Dropout uncertainty — ±1σ bands via 20 stochastic forward passes
🔬

Structure Integration

  • Interactive 3D viewer — bundled 3Dmol.js (offline); pLDDT, hydrophobicity, charge, SASA, chain, SS colour modes
  • H-bond & contact overlays — user-selectable colour, radius (H-bonds) and opacity (contacts)
  • SS Bead Model — linear bead diagram coloured by helix/sheet/coil from PDB HELIX/SHEET records
  • SASA profile — per-residue RSA/ASA via Shrake-Rupley; Ramachandran φ/ψ; distance map
  • One-click fetch — AlphaFold EBI, RCSB PDB; all chains selectable
🔗

External Databases

  • Pfam, ELM — domain and linear motif annotations overlaid on graphs
  • DisProt, MobiDB — experimental and consensus disorder annotations
  • PhaSepDB, IntAct — phase separation and protein interaction data
  • UniProt variants — natural variants and clinical pathogenicity data
  • AlphaMissense — per-position pathogenicity heatmap (Google DeepMind)
  • ESM2 Variant Effect Map — log-likelihood ratios for all 20×L substitutions

Installation

Up and running in three commands

Classical analysis works out of the box. Add the optional PyTorch + ESM2 block to unlock all 24 AI prediction heads.

Python ≥ 3.10 macOS Windows Linux ~200 MB

AI heads require torch ≥ 2.0 and fair-esm.

See the full installation guide for Linux Qt library setup and Windows-specific notes.

# 1. Create a conda environment
$ conda create -n beer python=3.12 -y
$ conda activate beer

# 2. Clone and install
$ git clone https://github.com/chemgame/BEER.git
$ cd BEER && pip install .

# 3. Launch
$ beer

──────────────────────────────────────────
# Optional — AI prediction heads (ESM2)
$ pip install "torch>=2.0" \
    --index-url https://download.pytorch.org/whl/cpu
$ pip install fair-esm scipy
